에이전틱 DB 진단

추출·메타데이터를 바탕으로 로컬 LLM이 읽기 전용 맥락만 분석합니다. 진단 이력은 같은 로컬 SQLite(connections.db)에 저장됩니다.

← 연결 · 추출

추출된 DDL·샘플·용량 메타데이터를 바탕으로 로컬 LLM(Qwen 2.5/3 계열, Ollama·vLLM OpenAI API)이 2단계 분석 후 진단 리포트를 생성합니다. RTX 5070 등 로컬 GPU에 맞는 양자화 모델을 Ollama에 올려 두면 됩니다.

데이터 안전: 진단(점검) 기능은 프로덕션 DB에 쓰기(INSERT/UPDATE/DELETE/DDL)를 하지 않습니다. 옵션 “실시간 용량 스냅샷”은 읽기 전용 시스템 카탈로그/용량 조회(SELECT)만 실행합니다. 리포트는 로컬 SQLite·화면에만 저장되며, LLM은 텍스트 생성만 합니다. (대용량 모델 실행으로 인한 GPU/CPU·메모리 부하는 발생할 수 있습니다.)
Ollama 미확인

원격/공용 서버에서는 OLLAMA_ALLOW_WEB_START=0 로 웹에서의 ollama serve 자동 기동을 끌 수 있습니다.
